# SOGL — Kommissionens forordning (EU) 2017/1485

System Operation Guideline. Fastlægger ensartede krav til drift af det europæiske transmissionsnet. Implementeret af Energinet som dansk TSO.

## Fem hovedområder

| # | Område | Artikler | Kerneindhold |
|---|--------|----------|--------------|
| 1 | Driftssikkerhed | 18-41 | N-1, operationel sikkerhed, koordinering |
| 2 | Driftsplanlægning | 42-113 | Year-ahead, month-ahead, day-ahead planlægning |
| 3 | LFC og reserver | 118-189 | FCR, FRR, RR dimensionering og aktivering |
| 4 | Driftstilstande | 18-25, 141 | Normal, alert, emergency, blackout, restoration |
| 5 | Data og information | 40-53 | Udveksling mellem TSO'er, ENTSO-E, RSC |

## Driftstilstande (art. 18)

- **Normal** — N-1 opfyldt, frekvens og spænding i bånd
- **Alert** — N-1 ikke sikret, afhjælpning under planlægning
- **Emergency** — operationelle grænser overskredet, akutte indgreb
- **Blackout** — spændingstab i et eller flere områder
- **Restoration** — genoprettelse efter Network Code on Emergency and Restoration (NC ER)

## Synkronområder og DK

- **DK1** (Vestdanmark) — del af Continental Europe (CE) synkronområde
- **DK2** (Sjælland og Lolland-Falster) — del af Nordic synkronområde
- Grænseflader mellem dem via Storebælt HVDC (600 MW LCC)

## Kerne-artikler

- **Art. 35** — Midlertidig fravigelse af N-1 i Alert
- **Art. 76-81** — TSO-TSO operationel koordinering
- **Art. 118** — Reserver overordnet
- **Art. 141** — Operationelle driftstilstande
- **Art. 153-164** — FCR, aFRR, mFRR dimensionering

## Implementering i DK

SOGL er direkte gældende forordning. Energinet udmønter i:

- Netreglerne (bindende regelsæt udstedt efter EFL § 26 stk. 3)
- Tekniske Forskrifter (TF 3.2.1 til 3.3.1)
- Operationelle procedurer i kontrolrum

## Nøglebegreber

- **CNEC** — Critical Network Element and Contingency (kritiske netelementer under kontingenser)
- **RSC/RCC** — Regional Security Coordinator / Regional Coordination Centre (Coreso, Nordic RSC, TSCNET)
- **DACF/IDCF** — Day-Ahead / Intraday Congestion Forecast
- **Load-Frequency Control (LFC)** — kontrolblokke der balancerer frekvens
